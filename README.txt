INTRODUCTION
------------

Smart Content FunnelEnvy is a connector module to allow variatons and audiences available through FunnelEnvy’s API to be used as
conditions in Smart Content for Drupal.

Note: Use of this module requires an active subscription to Funnelenvy’s personalization and segmentation platform.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/smart_content_funnelenvy

 * To submit bug reports, feature suggestions or to track changes:
   https://www.drupal.org/project/issues/smart_content_funnelenvy


REQUIREMENTS
------------

This module requires the following modules:

 * Smart Content (https://www.drupal.org/project/smart_content)


INSTALLATION
------------

Install using composer: composer require drupal/smart_content_funnelenvy


CONFIGURATION
-------------

This module does not require any configuration. However, please note that the FunnelEnvy tag must be added to your site
in some manner.
